Experimental Page

A place to test experimental themes and to create subpages easily. The current theme applied to this page is style_exp_0.2.css and is on version 0.2 as of deployment. There is no extensive meta tags on this page to allow for higher focus on other key web elements.

This page is also made for the strict purpose of creating new subpages on roadtogrambys-communal.cc/catalog, which can easily be pushed through GitHub Desktop (which is campy7's or other GitHub contributor's lovely job!).


Catalog Boilerplate Editor

This is a simple HTML formatter/compiler that takes text and images. I created a simple Markdown-esq markup language (Markedlydowner) to encode for the RTGC-specific format through javascript. These formats are made to work with the this page's style (until it's entirely overhauled).

Please include images as outlined in the syntax rules or else they will be entirely broken when pushed to the main branch.

Content encoded into HTML appears here. Copy and paste the result into a .txt file.
